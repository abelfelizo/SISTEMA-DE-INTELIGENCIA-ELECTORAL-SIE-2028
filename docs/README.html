Guía rápida

Uso en GitHub Pages: sube esta carpeta tal cual (contenido de SIE_2028_FULL) a un repositorio y activa Pages.

Encuestas sin programar

  1. Edita /data/encuestas_master.xlsx.
  2. Exporta a JSON con el nombre polls.json.
  3. Reemplaza /data/polls.json en el repo y haz push.

La web lee /data/polls.json. El Excel es tu editor.
